#To-Do Website
This is a simple To-Do website created using HTML, CSS, and JavaScript. The website allows you to create tasks, edit them and delete them.

##Features
Create tasks
Edit tasks
Delete tasks

##Usage
To use this website, simply open the index.html file in your web browser. You will see a form where you can enter a task. Once you have entered a task, click the "Add Task" button to add it to the list.

To edit a task, click the edit button next to the task. To delete a task, click the exit button next to the task.
